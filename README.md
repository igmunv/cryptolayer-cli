# CryptoLayer


Консольное приложение для библиотеки cryptolayer-base


## 🚀 Быстрый старт

```bash

git clone https://github.com/igmunv/CryptoLayer.git

cd CryptoLayer

git submodule update --init --recursive

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python3 run.py

```
